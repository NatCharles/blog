Drupal for dummies

Many, many people support the Drupal community by contributing code, answering queries (often within hours) and writing tutorials. However, there are often gaps in explanations, leaving users floundering or frustrated. Assumptions are made about the level of competence of users. The following miscellaneous tips are for the less technically-minded.

Hyphens in php code

I recently created a Bulk Export module (tutorial coming shortly) which wouldn't load. I got a 500 - Internal Server Error when I tried to enable it. The reason? I had put hyphens in the module name, which generated hyphens in the .inc and .module files.PHP cannot parse hyphens. When I replaced all hyphens with underscores, everything was ok. The trouble is, there is no documentation about this.
