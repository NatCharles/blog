Miscellaneous tips

21 March 2013

Here, I'll add tips on anything and everything web-related for Windows users. (I don't have a Mac and although I can and do use Linux, I find it a bit scary). I don't have a comments system in place, that'll come later, so if you have any comments, please email me. Any comments of your I publish, will of course be attributed to you.

TarTool

Every now and then you will download a 'zipped' file. You will need to 'unzip' it in order to extract the individual folders and files compressed within it. If this file has the suffix .zip, you can right-click on the file name, then select Extract All… and follow the prompts.

If, however, the file has the suffix .tar.gz, you'll need a program or tool to unzip it in Windows. There are downloadable programs you can purchase, but I found this neat little free tool . This is how you use it:

  • Download the zipped file TarTool.zip from the link above into your Downloads folder. Unzip it by right-clicking on the the file name and selecting Extract All…. Using the default options will give you 2 files - ICSharpCode.SharpZipLib.dll and TarTool.exe in the folder TarTool.
  • Now this is an important step - copy these two files into the folder that contains your tar.gz file.
  • Go up one level to the containing folder. Press shift and right-click, then select Open command window here. This generates a little black popup window. At the blinking cursor, enter TarTool.exe to view the options.
  • Enter TarTool yourFilename.tar.gz temp, where yourFilename is the name of the your zipped file and temp is the name of the folder you wish to contain the extracted files. Try it!

Note: An easy way to paste a filename into the command window is to first copy it in the usual way. Then when you are in the command window, right-click at the point you want to paste the filename and select paste.

Emmet

I'm a great fan of Open Source software. I already use Notepad++as a text editor. Although not the most sophisticated or comprehensive, it is sufficient for my needs. Via my Twitter feed, I discovered Emmet, 'the essential toolkit for web-developers'. I didn't realise that this is a plugin already embedded in Notepad++. Just go to Plugins/Emmet and it's right there. There are all sorts of options to help speed up your editing. For comprehensive and easy-to-follow documentation, go to this page. Here is an Emmet cheat sheet.

My favourite commands are:

  • Expand Abbreviation (keyboard shortcut Ctrl+Alt+Enter). Entering 'em' for example, will generate the code <em></em>.
  • To highlight a block of text between a pair of matching tags, click somewhere inside the block, then select Match pair inward. To include the tags as well, click within one of the tags, then select Match pair inwards.Having highlighted your text, you can either move or delete it.

IRC (Internet Relay Chat)

There's an excellent video here that describes what IRC is, why it might be useful to you and how to get set up. Here are two methods:

Webchat

Webchat is a popular web-based IRC client , based on the freenode network. You don't need to download any software. First, you will need to decide what channel you want. This page has a comprehensive list. You might want to narrow your search by googling 'irc whatever'. Here is a list of Drupal IRC channels. Next, choose a nickname. You can register this nickname if you want to. Then click on the Webchat.freenode.net, enter your nickname and the captcha. You will then be taken to the webchat window.

Your cursor will be focused at a text field at the bottom of the window. This is where you enter your chat.

Webchat commands

Here are some links to webchat commands - Basic IRC Commands and Common IRC Commands. What these sites don't tell you is that you enter these commands in this same text field. Commands start with the character '/'. For example, '/msg NickServ INFO nick' (where nick is your nickname) will give you information about this nick.

ChatZilla

ChatZilla is a Firefox addon, which is a bit more user-friendly. It lets you search for both IRC networks (eg. Freenode) and channels within that network. The commands, however, are the same as for Webchat.

NetBeans

If you have your default configuration set to Chrome with NetBeans Integration and you also have 'Google Chrome to Phone Extension' installed and enabled, you get this message in your NetBeans Output pane: Port error: Could not establish connection. Receiving end does not exist. The only way I found to stop this message was to disable this extension.
